# Image Files

This directory contains image assets for the Snap Cadence application.

## Current Files

- ✅ `football.svg` - American football SVG graphic (placeholder)

This SVG file displays during the SNAP_ACTION state (when the snap occurs).

## Production Preparation

For production deployment, you may want to replace the placeholder SVG with:

1. A high-quality PNG or SVG of an American football
2. Dimensions: Recommended 200x200px or larger
3. Transparent background preferred
4. Optimized file size for web

The current SVG placeholder is a simple brown football with laces and stitching.
